# Start using Sovereign

You use Sovereign through an instance run by an operator. The Sovereign project
does not create a global account that automatically grants access to every
instance.

## 1. Find an instance

Open the [instance directory](/instances). Review the intended audience, access
policy, operator, installed apps, and last verification date.

## 2. Review the operator

The operator controls accounts, policies, infrastructure, apps, backups,
retention, and support. Read the operator's privacy and service terms before
creating an account or adding data.

## 3. Obtain access

Registration may be open, approval-based, invite-only, or limited to a
demonstration. Follow the policy shown for that instance. Do not assume a visible
registration form guarantees access or a particular service commitment.

## 4. Enter the workspace

After signing in, the Launcher shows the apps available to you. Account settings
cover your profile and security options. Other apps and permissions depend on
the operator's configuration.

## 5. Install it as an app

Install the instance as a PWA to give it a Home Screen, app launcher, or desktop
icon and open it in a standalone window. An installed app can also receive push
notifications when the operator and device support them.

[Follow the PWA installation and notification steps](/docs/pwa)

## 6. Ask the operator for support

Account access and instance data are controlled by the operator. Use the support
channel listed for that instance rather than contacting an unrelated operator or
the Sovereign project.
