# User documentation

Sovereign user documentation begins with the instance relationship because
accounts, apps, policies, and support are controlled by each operator. This page
routes to canonical guides rather than repeating their instructions.

## Start here

- [Find and evaluate an instance](/get-started/users)
- [Review listed instances](/instances)
- [Install an instance as a PWA and enable push](/docs/pwa)
- [Learn about the Account app](/plugins/account)
- [Learn about the Launcher](/plugins/launcher)
- [Read the security model](/security)

Detailed end-user guides for each installed app should be provided by that app
and linked by the operator. App availability differs between instances.
