# Find a Sovereign instance

A Sovereign instance is operated independently. The operator controls accounts,
policies, installed apps, retention, backups, infrastructure, and support. A
listing means the endpoint has been identified; it is not an audit, endorsement,
availability guarantee, or service commitment from the Sovereign project.

Review the operator's policies before registering or adding data.

## Directory policy

The first directory is manually curated. Before an entry is presented as a
verified hosted service, it must provide:

- A public operator identity and canonical HTTPS URL.
- A clear access policy: open, approval required, invite only, or demonstration.
- The intended audience and confirmed available apps.
- Privacy and support links, plus terms when applicable.
- A recent manual verification date.

Automated discovery, health monitoring, and unreviewed submissions are outside
the initial directory scope.
